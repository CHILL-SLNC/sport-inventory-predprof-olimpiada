Инструкция по установке и развертыванию:
1.- Установите Docker Desktop и зарегистрируйтесь.

2.- Внутри Pycharm откройте папку проекта.

3.- Заполните файл .env в соответствии с инструкцией по заполнению (*).

4.- Проверьте,чтобы порты 5173, 8000, 3306 были свободны от запущенных процессов.

5.- Внутри вкладки "Terminal" введите в консоль следующую команду:
docker login -u <Ваш логин Docker при регистрации>

6.- Введите пароль от аккаунта Docker, после чего при успешной авторизации введите команду:
docker-compose up --build

Теперь по ссылке http://localhost:5173/ вам доступно наше веб-приложение, а по адресу http://localhost:<BACK_PORT>/ доступно API этого приложения.

* Инструкция по заполнению файла .env:
DB_NAME - Имя базы данных в СУБД MySQL для хранения информации приложения.
DB_HOST - Наименования хоста базы данных (При запуске через Docker не рекомендуется менять значение по умолчанию).
DB_PORT - Порт, на котором будет доступна БД (Не рекомендуется менять значение по умолчанию).
DB_USER - Имя пользователя для базы данных (Не рекомендуется менять значение по умолчанию).
DB_PASSWORD - Пароль для доступа к базе данных.
FRONT_PORT - Порт, на котором доступен сайт приложения (Не рекомендуется менять значение по умолчанию).
SECRET_KEY - Секретный ключ шифрования JWT-токенов.
ALGORITHM - Наименования алгоритма шифрования (Не рекомендуется менять значение по умолчанию).
ACCESS_TOKEN_EXPIRE_MINUTES - Время жизни access токена в минутах.
ADMIN_USERNAME - Логин администратора для создания учетной записи при запуске приложения.
BACK_PORT - Порт, на котором доступно API приложения (Не рекомендуется менять значение по умолчанию).
ADMIN_FIRST_NAME - Имя администратора.
ADMIN_LASTNAME - Фамилия администратора.
ADMIN_SECOND_LASTNAME - Отчество администратора.
ADMIN_PASSWORD - Пароль для доступа к учетной записи администратора
